Package Name:
fastn-community.github.io/pretty-ws

Overview

promo-banner is a new conversation experience via using FTD lang. To use promo-banner on your fastn package, add below into lines FASTN.ftd file:
-- fastn.dependency: fastn-community.github.io/pretty-ws
-- fastn.auto-import: fastn-community.github.io/pretty-ws/lib as lp
Lang:
ftd

Invocation:

-- lp.promo-banner: At vero eos et accusamus et 
cta-label: Sign me up
src: $assets.files.invoicing.png
cta-url: https://fastn-community.github.io/pretty-ws/

Properties:

Below are details of all properties of this component.
title
This is a default property of the promo-banner component. It’s used to add a title to promo-banner component.
cta-label
This is a default property of the promo-banner component. It’s used to add a click to action button title.
cta-url
This is a default property of the promo-banner component. It’s used to add a link to click to action button.
src
This is a default property of the promo-banner component. Using src attribute image can be added.
