Properties:

Below are details of all properties of this component.
title
This is a default property of the promo-banner component. It’s used to add a title to promo-banner component.
cta-label
This is a default property of the promo-banner component. It’s used to add a click to action button title.
cta-url
This is a default property of the promo-banner component. It’s used to add a link to click to action button.
src
This is a default property of the promo-banner component. Using src attribute image can be added.
